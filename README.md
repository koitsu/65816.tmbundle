65816.tmbundle
==============

This is a TextMate/Sublime Text bundle for WDC's 65816 assembly language.  Some effort is taken to be 
slightly intelligent about some opcode/operand pairs.

Currently supported:
* Generic 65816 (mnemonics only)
* ORCA/M Assembler
* MPW IIgs Assembler
* Merlin 8/16+ Assembler
* NinjaForce Assembler
* EDASM Assembler (6502)
